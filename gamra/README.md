# Agri Boutabssil POS

Professional Agricultural Point of Sale and inventory management system.

## Setup
1. Accept Firebase terms in the UI.
2. The application will automatically provision the database.

## Features
- POS with Cart
- Inventory Management (with Buy/Sell prices)
- Customer/Supplier Debt Tracking
- Staff Management (Admin/Staff roles)
- Multilingual (AR, FR, EN)
- Real-time updates with Firebase
